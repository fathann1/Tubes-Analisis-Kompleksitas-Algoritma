Judul: Analisis Kompleksitas Algoritma Pembacaan dan Pengolahan Data Sensor Suhu dan Kelembapan DHT22 pada Sistem Monitoring Berbasis Mikrokontroler
Deskripsi: Tugas Besar ini bertujuan menganalisis dan membandingkan kompleksitas waktu dan ruang antara algoritma pembacaan data sensor suhu dan kelembapan DHT22 yang menggunakan pendekatan iteratif dan rekursif. Selain itu, penelitian ini juga bertujuan untuk mengevaluasi efisiensi masing-masing pendekatan dalam hal kecepatan eksekusi, penggunaan memori, serta kestabilan sistem selama proses pembacaan data. Melalui analisis tersebut, diharapkan dapat ditentukan pendekatan algoritmik yang paling optimal untuk diterapkan pada sistem monitoring suhu dan kelembapan berbasis mikrokontroler, terutama pada perangkat dengan keterbatasan sumber daya komputasi.
